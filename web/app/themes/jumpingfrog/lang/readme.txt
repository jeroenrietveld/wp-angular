In this folder language files (.po and .mo extensions) can be placed to translate this theme.
The text-domain used in this theme is 'jumpingfrog'.

File names must be the equivalent of the WPLANG value in the wp-config (like 'en_EN' or 'nl_NL').
This value is a combination of the language code [1] and country code [2].

Language files can be generated and translated using the program Poedit.
